## ✨ Advanced Features & Engineering

This project goes beyond standard web design, incorporating UI techniques used by elite tech companies (Vercel, Stripe, Linear):

*   **The "Live Data Core":** A custom-built, continuous SVG + Framer Motion animation visualizing a Node.js server interacting with a database, client, and WebSockets.
*   **Magnetic Call-to-Actions:** Buttons utilize spring physics to intelligently pull toward the user's mouse cursor, creating premium tactile feedback.
*   **Vercel-Style Spotlight Cards:** Project cards feature a dynamic, mouse-tracking radial gradient "flashlight" effect on hover.
*   **Aurora Typography:** The hero section features a continuously flowing liquid gradient text animation.
*   **Advanced Fluid Cursor:** A custom magnetic trailing cursor that morphs and expands when interacting with clickable elements.
*   **3D Tilt Physics:** Skill cards track mouse X/Y coordinates to tilt dynamically in 3D space.
*   **Centralized Data Architecture:** All text, projects, and skills are decoupled from the UI and managed via a single, easily updatable `data.ts` file.
